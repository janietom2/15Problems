Find All Duplicates in an Array

Given the coordinates of four points in 2D space, return whether the four points could construct a square.

The coordinate (x,y) of a point is represented by an integer array with two integers.

Code
                            
                                class Solution:         
                                def findDuplicates(self, nums):
                                    repeated = []
                                    seen = []
                                    
                                    for x in nums:
                                        if x not in seen:
                                            seen.append(x)
                                        else:
                                            repeated.append(x)
                                    
                                    return repeated
IDEAL

This is a simple problem for getting which numbers are repeated in an array/list.

No need of explaining the only thing I did is create to list one for seen values and the other for repeated, then check if the value of the given array/list is in seen if not means they are repeated then added to the other array/list
