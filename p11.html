Compare Version Numbers

Compare two version numbers version1 and version2.

If version1 > version2 return 1; if version1 < version2 return -1;otherwise return 0

Compare two version numbers version1 and version2. If version1 > version2 return 1; if version1 < version2 return -1;otherwise return 0.

You may assume that the version strings are non-empty and contain only digits and the . character. The . character does not represent a decimal point and is used to separate number sequences. For instance, 2.5 is not "two and a half" or "half way to version three", it is the fifth second-level revision of the second first-level revision.

Code
                            
                                class Solution:
                                def compareVersion(self, version1, version2):
                                    """
                                    :type version1: str
                                    :type version2: str
                                    :rtype: int
                                    """
                                    
                                    v1 = len(version1)
                                    v2 = len(version2)
                                    
                                    max_l = max(v1, v2)
                                    min_l = min(v1, v2)
                                    
                                    for i in range(max_l):
                                        if i >= min_l :
                                            return 1
                                        if version1[i] == "." or version2[i] == ".":
                                            continue
                                        if version1[i] != version2[i]:
                                            return -1
                                        
                                    return 1
IDEAL

This problem is interesting because it is basically doing same on other problems with a long string that need to compare each other to check if the version is the same. This works for checking versioning in software. The application of this I can think of is that a software installation package check for the version before installing in order to be compatible or to avoid any trouble for the user. This is very useful

The solution i found is basically count the string lenght (get max and min) then on a loop going through each character and then iterate with a limit of the min length of the two version strings, then compare if the strings are the same, if not then return -1, if everything is ok then return 1
