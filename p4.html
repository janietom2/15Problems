Search a 2D Matrix

Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:

* Integers in each row are sorted from left to right.
* The first integer of each row is greater than the last integer of the previous row.

Code
                            
                                class Solution:
    
                                def binarySearch(self, alist, value):
                            
                                    first = 0
                                    last = len(alist)-1
                                    found = False
                                    while first<=last and not found:
                                        midpoint = (first + last)//2
                                        if value == alist[midpoint]:        
                                            found =  True 
                                        else:
                                            if value < alist[midpoint]:
                                                last = midpoint-1
                                            else:
                                                if value > midpoint:
                                                    first = midpoint+1  
                                    return found
                                
                                def searchMatrix(self, matrix, target):
                                    linear_array = []
                                    
                                    for x in range(len(matrix)):
                                        for y in range(len(matrix[x])):
                                            linear_array.append(matrix[x][y])
                                    
                                    return self.binarySearch(linear_array, target)
IDEAL

The problem is a basic search in a 2d array. The thing is that the numbers are already sorted.

My solution might not be the best but by the fact that they are sorted I use binary search to find if the key was in there. However the binary search only support 1D arrays/list so what I did is that I took the 2D array passed all values to a 1D array
