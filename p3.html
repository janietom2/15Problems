Multiply Strings

Given two non-negative integers num1 and num2 represented as strings, return the product of num1 and num2, also represented as a string.

Code
                            
                                class Solution:
                                def strToInt(self,s):
                                    num = 0
                                    for ch in s:
                                        n = ord(ch) - ord('0')
                                        num = num*10 + n
                                    return num
                            
                                
                                def multiply(self, num1, num2):
                                    mult = self.strToInt(num1) * self.strToInt(num2)
                                    
                                    return mult
IDEAL

The problem gives two strings with numbers that are needed to be multiplied. For example we have 2x3=6

The main idea here is to create a method hat converts string to integer by using their Unicode code then getting the number. Python has a method name ord() which returns the Unicode code. I searched on Google how to get an integer using unicode and a answer was added the unicode of X string by the unicode of 0 then that would be the integer then multipliying it by 10, since its a decimal number. In this case a loop is used because the number can go from single to multiple like 100 or 1000 (Limit of Integer 2^31)
