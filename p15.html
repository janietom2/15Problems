Daily Temperatures

Given a list of daily temperatures T, return a list such that, for each day in the input, tells you how many days you would have to wait until a warmer temperature. If there is no future day for which this is possible, put 0 instead.

Code
                            
                                class Solution:         
                                    def dailyTemperatures(self, T):
                                    nxt = [float('inf')] * 102
                                    ans = [0] * len(T)
                                    for i in xrange(len(T) - 1, -1, -1):
                                        #Use 102 so min(nxt[t]) has a default value
                                        warmer_index = min(nxt[t] for t in xrange(T[i]+1, 102))
                                        if warmer_index < float('inf'):
                                            ans[i] = warmer_index - i
                                        nxt[T[i]] = i
                                    return ans
IDEAL

This is an array list problem that is very similar to a problem we did on the exam about the list of stocks, it was about when to sell by checking element in the list/array from a item and forward moving. This is basically almost the same problem by

The solution is iterate the the list while getting the element into a variable then check to the next until the end of the array. While iterating to the next do a difference (by substraction) to get the difference between a number to another (temp) then added to an array.
