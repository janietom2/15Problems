Longest Substring Without

Given a string, find the length of the longest substring without repeating characters.

Code
                            
                                    seen = {}
                                    current_max = result = 0
                                    for i, c in enumerate(s):
                                        if c not in seen or seen[c] < (i- current_max):
                                            seen[c] = i
                                            current_max += 1
                                        else:
                                            current_max = i - seen[c]
                                            seen[c] = i
                                        result = max(result, current_max)   
                                    return result
IDEAL

This problem counts the substring of a given string. For example, abcabcbb will count only 3 since the longest substring is abc. This problem can be solved by using a "seen" array or a hashmap for already seen (by iteration) characters. Also a counter that counts the streak of character without repeating until it sees a repeated (already seen) character.

This problem certainly looks alike problems we did in class using hashmaps/hastables to check if some values have been seen.
