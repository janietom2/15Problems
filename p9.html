Reverse Words in a String

Given an input string, reverse the string word by word.

Code
                            
                                class Solution:
                                def restoreIpAddresses(self, s):
                                    """
                                    :type s: str
                                    :rtype: List[str]
                                    """
                                    
                                    words = []
                                    curr_w = ''
                                    
                                    # get words
                                    for i in range(len(s)):
                                        if i == len(s)-1:
                                            curr_w += s[i]
                                            words.append(curr_w)
                                        
                                        if s[i] == " ":
                                            words.append(curr_w)
                                            curr_w = ''
                                        else:
                                            curr_w += s[i]
                                    
                                    final_w = ''
                                    
                                    for x in range(len(words)-1, -1, -1):
                                        final_w += words[x]
                                        
                                        if x != 0:
                                            final_w += ' '
                                    
                                    return final_w
IDEAL

This problem was one of the first we saw in class, however I found out that my code was wrong when I wrote in a notebook paper sheet. I noticed that the problem is that the last word will not get into the array since the " "(space) will avoid getting that word. Anyway, this problem tries to get each word and put it on reverse creating a non-sense setence.

The solution I found (a new one) is counting or checking the ending of the setence because this is problems need to check the ending because the part where the word is going to be added into an array is when a " "(space) is detected, hence the last word will not have a space to the right, so need to check if it is the last word. Then after getting each word into an array/list, need to traverse (again) that array reversed (starting from the lenght of it to 0) then concatinate a string variable and adding the word and a space to the right. This also has a trick since the last word do not need a space to the right, so again need a condition to do not include it at the end.

Same with problem p8 with the IP, there are things that need to be detected to format the string depending on what the problem needs. This was a good follow up to the other problems of strings.
