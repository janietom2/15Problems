Remove Nth Node From End of List

Given a linked list, remove the n-th node from the end of list and return its head.

Code
                            
                                class Solution:
                                def removeNthFromEnd(self, head, n):
                                    """
                                    :type head: ListNode
                                    :type n: int
                                    :rtype: ListNode
                                    """
                                    nodes = []
                                    while head:
                                        nodes.append(head.val)
                                        head = head.next
                                    
                                    del nodes[-n]
                                    
                                    return nodes
IDEAL

This problems needs to remove a place in the node (which is given by parameter) starting for the end of a list of nodes.

This problem is really simple for a point of view. The list can be iterated using a loop or by recursion then point using a counter until reaches n then point that to null.

However I believe there is a more simple solution using what we have on Python. Lists are very useful and since the problem needs a list (return) the values can be passed on a list then use the del function of the lists can delete the node n then return the list

This problem helps in other problems I personally have seen in class like reversing a linked list, the lists can help a lot, however the running time can be a problem
